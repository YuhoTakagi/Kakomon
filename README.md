過去問共有アプリ
これは、大学の過去問を共有し、AIによる類題作成や質問ができる学習支援プラットフォームです。Next.js (App Router) と shadcn/ui を使用して構築されています。

✨ 主な機能
本プロジェクトは、学生が過去問にアクセスしやすくし、より深い学習をサポートすることを目的としています。
過去問の閲覧: 学部 → 学科 → 科目 → 教授 の階層で、投稿された過去問を簡単に検索・閲覧できます。
過去問の共有: 自分が持っている過去問をアップロードし、他の学生と共有できます。（現在はデモのためテキスト入力）
AIによる類題作成: 登録された過去問をベースに、OpenAI API (gpt-4o-mini) を利用して新しい類題を自動生成します。
質問機能: 特定の過去問に対して質問を投稿できます。（モック）
認証: ユーザー登録、ログイン、アカウント管理機能。（現在は lib/mock-data.ts によるモックデータで動作しています）
APIキー設定: 類題作成機能を使用するために、ユーザーが自身のOpenAI APIキーを設定画面から登録できます。（ローカルストレージに保存）
